الأسبوع ١٠: Scratch — التحكّم بالكائن

الصف التاسع (١٤–١٥ سنة) — حركة مفاتيح/حدود/ارتداد، سرعة/تسارع بسيط، أنيميشن بالمظاهر، اصطدام وعداد نقاط.

🎯 الأهداف التعليمية

تطبيق أنماط تحكّم قياسية، ضبط حدود المنصّة، بناء أنيميشن بسيط، كشف اصطدام، وإدارة نقاط.

🧪 محطات العمل (45–50 د)

Keys، Borders، Animation، Collision، HUD/Score، Debug/Export — ٥–٧ دقائق لكل محطة.

🧭 خطة الزمن (٤٥–٥٠ د)

تهيئة → محطات → شرح موجز → تقويم سريع.

🧠 الشرح والتوسّع

سرعة/تسارع بسيط، قَص وتمحور الحركة، تحسين تجربة اللعب.

🛡️ السلامة والإتاحة

حسابات المدرسة، حفظ محلي، واتّباع WCAG.

📝 التقويم داخل الحصة

Quiz (٨ بنود)، ورقة مهام، Mini-Project: تحكّم + اصطدام + نقاط.

📦 الموارد والروابط

Scratch (ويب/أوفلاين)، Sprites/Costumes بسيطة، أصوات اصطدام.
